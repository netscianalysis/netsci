NetSci: A Toolkit for High Performance Scientific Network Analysis Computation
==============================================================================

:Release: |release|
:Date: |today|

**NetSci** NetSci is a specialized toolkit designed for advanced network analysis in computational sciences. Utilizing the capabilities of modern GPUs, it offers a powerful and efficient solution for processing computationally demanding network analysis metrics while delivering state-of-the-art performance.
